# pca-protocols.md — Personal Context Artifact #4: Hard Rules

> **What this file is:** your non-negotiables. Rules that don't bend. Agents treat violations as P0 errors, not soft preferences.
>
> **Who reads it:** every agent that takes action on your behalf. Especially: Executive Assistant, CEO orchestrator, email drafter, calendar optimizer.
>
> **When you update it:** when a hard rule is violated → either the rule needs better enforcement OR the rule was wrong (write a new one). Keep this SHORT — if it's getting long, you're adding soft preferences that belong in `pca-mental-models.md`.
>
> **Companion files:** `pca-wiki.md`, `pca-mental-models.md`, `pca-voice.md`.
