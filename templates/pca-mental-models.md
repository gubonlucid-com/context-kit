## Decision Framework (for hard calls)

When facing a genuinely hard decision, I run it through these in order:

1. **Is this reversible?** If yes, move fast. If no, slow down.
2. **Whose decision is this?** If mine, decide. If someone else's, surface don't solve.
3. **What would I regret more — doing it or not doing it?** The regret test for close calls.
4. **Does this fit my trade-off priors above?** If it conflicts, surface the conflict explicitly.
